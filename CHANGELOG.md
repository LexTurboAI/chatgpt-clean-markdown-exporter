# Changelog

## 0.1.0 - 2026-07-08

Initial public release.

- Added standalone Tampermonkey/Violentmonkey userscript.
- Added floating `MD` export button.
- Added menu command: `Export Clean Markdown`.
- Added clean Markdown conversion for ChatGPT conversation JSON.
- Replaced uploaded and generated images with explicit placeholders.
- Avoided Base64/data URI/sediment image embedding.
- Removed external dependencies: no JSZip, no html2canvas, no `@require`.
- Added English and Polish documentation.
- Added static PowerShell safety check.

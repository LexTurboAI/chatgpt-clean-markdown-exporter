# Instalacja w Tampermonkey

1. Zainstaluj Tampermonkey albo Violentmonkey.
2. Otwórz plik `ChatGPT-Clean-Markdown-Exporter.user.js`.
3. Wybierz instalację skryptu.
4. Wejdź w konkretną rozmowę ChatGPT.
5. Kliknij pływający przycisk `MD`.

Jeżeli przycisk się nie pojawia:

- odśwież rozmowę;
- sprawdź, czy Tampermonkey jest aktywny na `chatgpt.com`;
- sprawdź, czy jesteś zalogowany do ChatGPT;
- otwórz konkretny czat z adresem zawierającym `/c/`.
